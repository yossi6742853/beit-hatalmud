# בית התלמוד - מערכת ניהול מוסד חינוכי

מערכת ניהול מוסד חינוכי מבוססת GitHub Pages.

## מאפיינים
- SPA עם routing מבוסס hash
- ממשק עברי מלא (RTL)
- מותאם למובייל
- מצב כהה/בהיר
- כניסה עם PIN
- מטמון localStorage
- עובד אופליין עם נתוני דמו

## דפים
- לוח בקרה - סטטיסטיקות וגרפים
- תלמידים - רשימה, חיפוש, כרטיס תלמיד
- נוכחות - סימון יומי עם קיצורי מקלדת
- צוות - רשימת אנשי צוות
- כספים - מעקב שכר לימוד
- טלפון - חייגן עם אנשי קשר
- מצלמות - צפייה במצלמות המוסד
- הגדרות - הגדרות מערכת

## טכנולוגיות
- Bootstrap 5.3.2 RTL
- Bootstrap Icons
- Google Fonts (Heebo)
- Chart.js
- Vanilla JavaScript

## התקנה
1. העלה את הקבצים ל-GitHub repository
2. הפעל GitHub Pages מתוך Settings
3. הזן את כתובת ה-API של Apps Script בהגדרות

כל הזכויות שמורות - בית התלמוד 2026
